sqlBackups

Description

Backup Shell Script for local MySQL server

Installation

Copy the file in /root/backup_sql.sh

Set execution rights chmod +x /root/backup_sql.sh

Add a cron job for this backup crontab -e

Check your backups in /var/backups/sql/*

Share it, fork it and improve it.
